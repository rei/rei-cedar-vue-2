Navigation Lists

Make navs used as navigation accessible

If you are using navs to provide a navigation bar, be sure to add a role="navigation" to the most logical parent container of the <ul>, or wrap a <nav> element around the whole navigation. Do not add the role to the <ul> itself, as this would prevent it from being announced as an actual list by assistive technologies.

Sorting example

This example demonstrates a sort list using radio buttons. To accomplish this we add a few more atomic parts to our list-group items. We change the markup within the parent list items to match that of the radio markup from our radio example and add the radio class to the list-group-item. Then, to style the radio as a sort checkmark we add the appropriate toggle class of toggle-sort.

Filter Menu

Taking what we have built in the sort example we can add one more class: collapsibleList to have our radio or checkbox work as a control to expand and collapse our list groups. for this example we have used checkboxes and have changed the toggle-sort to toggle-expand-collapse.
