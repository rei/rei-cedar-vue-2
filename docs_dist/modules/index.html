UI Modules are groups of collections and elements joined together with behavior.

UI Modules are groups of collections and elements joined together with behavior.

Modules can consist of similar and/or disparate collection types.

Building up from collections to modules encourages creating standalone, portable, reusable components.

These include components like, tabs, modals, and tooltips.
