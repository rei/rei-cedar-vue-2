Aspect Ratio

Utilize the .utility-aspect-ratio mixin or .ratio-[ square, 1-2, 2-3, 3-4, 9-16, 2-1, 3-2, 4-3, 16-9 ] class to set an element with no implicit height ( such as the case when its contents are positioned absoluetely ).

Applying .content-[ top, center, bottom ] (or using modifier --content-[ top, center, bottom ] will align contents within the container.

Custom heading {% endhighlight %} {% highlight scss %} // Usage as a mixin div { .utility-aspect-ratio( 9/16; bottom ); } {% endhighlight %}
