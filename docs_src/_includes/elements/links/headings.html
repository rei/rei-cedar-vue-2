Heading Links

Use when an H1, H2, or H3 Header is presented as a link (Primarily used on Hub or Campaign-related pages).

Default state: Follows text color, and the class sets the color attribute to inherit.

Hover state: No underline, color set to light grey with the @overlay-link-color-active variable.

The icon-rei-right-arrow icon is set to 0.5em of the headings font size, this allows the icon to scale to the heading rather that adhering to a strict px value. This icon is set to the text baseline

Please note the lack of whitespace between heading text and icon span, this is intentional and necessary for correct spacing.
