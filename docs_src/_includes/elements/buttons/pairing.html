Use buttons in these combinations to communicate hierarchy of user action. The family (Core, Explore, ect) color tone represents the intended primary call(s) to action.

A singleton button possesses no margin

Buttons which have a previous sibling which is also a button receive a left margin however if your buttons will wrap prior to displaying block you will have an undesirable staircase. For this reason we offer the btn-group inline-group classes.
