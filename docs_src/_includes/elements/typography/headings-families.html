Headings

Core Headings

All HTML heading tags, <h1> through <h6>, are available. .h1 through .h6 classes are also available for core headings, when you want to match the font styling of a heading but still want your text to be displayed inline.

Explore/Impact Headings

Complementing the architectural type styles, these uppercase headline styles provide visual differentiation for storytelling elements. Apply the .heading-editorial class to any <h1> through <h6> to achieve the explore heading styles

{% include /markup-templates/typography/headings.html param="explore" %}
{% highlight html %}

H1 explore

{% endhighlight %}

Hero Headings

This family anchors large-scale mastheads with graphic or photographic backgrounds. Apply the .heading-hero class to any <h1> through <h6> to achieve the explore heading styles
