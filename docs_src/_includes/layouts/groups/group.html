Groups provide unique content grid and grid gutter options and also allow content containers such as card, tile and well various flex display options. Groups are intended to be homogenous rather than mixed.

Group grid

Like our Page Grid, Groups make use of columns for container separation and any gutter spacing requierments. These work in the same manner as the page grid columns and support display variation at the breakpoint level.

Group gutters

Groups also make use of gutters for container separation, however unlike our Page Grid, Group gutters can be collapsed or open and their delimiter remains constant across breakpoints.

Collapsed Group gutter

By default, Grid gutters are in a collapsed state. This is intended for usage where all content containers are related to a larger context and would be appropriate for wells or tiles.
