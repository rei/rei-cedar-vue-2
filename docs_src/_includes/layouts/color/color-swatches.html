Color Palettes docs

Core Palette

These core styles are used most frequently throughout REI.com. They have been adapted form our Brand Standards palette for interactive use. Be sure to check any undocumented background/foreground combinations for accessibility.

Background color of default button.

Background color of disabled button.

Background color of input group addons
Helper variable
This variable is mixed with several primary brand colors to create active, hover, and focus states.
Usage
Default background, typographic use on dark or photographic backgrounds, default buttons.

Helper variable
This variable is mixed with several variables to create active, hover, and focus states.
Usage
Limited typographic and structural elements, background fields, high contrast primary button.
Note:
the @background-black-ruleset(); is used by the overlay-dark-1 class to set background container color. The ruleset also turns text white.
